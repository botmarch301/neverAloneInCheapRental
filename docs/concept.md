## 장르 — 생활 시뮬레이션 + VN 연출

플레이어가 **일상 활동을 직접 관리**하면서 스토리가 진행됨.
핵심 장면에서 **VN 스타일 연출** (대사, 회상, 선택지)과
**아케이드성 인터랙션**, **성감 시뮬레이션**이 삽입.

### 일상 활동 (사용자 관리)

| 활동 | 설명 |
|------|------|
| **식사** | 체력/양기 유지에 영향 |
| **세신** | 위생/컨디션 관리 |
| **청소** | 집 상태 관리 |
| **휴식** | 체력 회복, VR 사용 시간 |
| **일** | 게임 회사 출근, 수입 확보 |
| **쇼핑** | 식량, 생활용품, VR 콘텐츠 등 구매 |

### 공포 인터랙션 (아케이드)

| 상황 | 조작 | 실패 시 |
|------|------|---------|
| **가위눌림** | 줄어드는 게이지를 **키 연타**로 채움 (고개를 돌리려는 발악) | 게이지 소진 → 공포 누적 |
| **인기척** | **좌/우 키**로 균형 조절 — 돌리고 싶은 마음 vs 돌리고 싶지 않은 마음 | 너무 일찍 돌림 → 귀신 목격 → 공포 대량 누적 |

### 성감 시뮬레이션 (4막~)

- 젖꼭지, 가슴, 성기 등을 **사용자가 직접 조작** (터치/클릭)
- **성감을 느끼기 좋은 곳**을 찾아 **적당한 리듬감**으로 터치하는 것이 핵심
  - 빠른 액션이 아닌, **느리고 적절한 리듬**이 중요
  - 올바른 부위 + 올바른 리듬 = 흥분 상승
  - 잘못된 부위 or 너무 빠름/느림 = 반응 약함
- 반복적 행위는 **미리 설정된 오토 기능**으로 전환 가능
- **숨겨진 흥분도** — 별도의 게이지 UI를 표시하지 않음

#### 시각적 흥분 피드백 (게이지 대체)

| 단계 | 시각적 변화 |
|------|-----------|
| 초기 | 표정 변화, 가슴이 부풀어 오름 |
| 중반 | 젖꼭지가 섬, 호흡 거칠어짐 |
| 후반 | 클리토리스·소음순 부풀어오름, 애액 증가 |
| 절정 | 몸 경련, 표정 변화, 절정 반응 |

> **핵심**: 플레이어가 **몸의 반응을 관찰**하며 흥분도를 파악해야 함.
> 숫자 게이지가 아닌 **시각적 변화**로만 상태를 알 수 있음.

> [!IMPORTANT]
> **디자인 원칙: 수치 비노출**
> 내부적으로 흥분도·공포도·양기 등의 수치가 존재하더라도,
> 플레이어에게는 **어떤 수치도 직접 보여주지 않는다**.
> 모든 피드백은 **시각적 변화와 연출**로만 전달.

## 주제의식

> **이승의 존재와 저승의 존재는 함께할 수 없다.**

귀신이 아름다워질수록 주인공은 죽어간다 — 한쪽이 살면 한쪽이 소멸하는 구조.
이 메시지는 대사가 아닌 **캐릭터 외형의 역방향 변화**로 전달된다.

```
귀신:  검은 그림자 (성별 불명)  →→→→→  아름다운 여성
주인공: 통통한 오타쿠          →→→→→  미라 같은 폐인
```

모든 엔딩에서 이 룰은 깨지지 않는다:
- **배드**: 함께했더니 산 자가 죽었다
- **노멀**: 함께할 수 없으니 죽은 자를 내쫓았다
- **트루**: 함께할 수 없으니 죽은 자가 스스로 떠났다 (성불)

## 로그라인

억울하게 죽어 악귀가 된 처녀귀신이 사는 집에 이사 온 내성적인 남자.
남자의 **양기(정액)**가 악귀의 **탁기**를 정화하면서 둘의 관계가 변해간다.

## 주인공

| 항목 | 설명 |
|------|------|
| **성별** | 남성 |
| **나이** | 20~40대 |
| **성격** | **히키코모리 모쏠 오타쿠** — 겁이 많고 유약하며 모질지 못함 |
| **특징** | 귀신이 무서워 VR을 구매 → VR 착용 중 귀신이 안 보임 + 자위가 일상 |
| **생활습관** | **전자 담배** 사용, **인스턴스/편의점 음식** 주식 → **집에서 불을 전혀 안 씀** |
| **페티쉬** | **모유 페티쉬** — 엄마가 일찍 죽어 젖을 물린 기억이 없음 (편부 가정) |

### 외형

| 부위 | 묘사 |
|------|------|
| **눈** | 처진 눈 + 짙은 다크서클 + 약간 크고 불안한 동공 |
| **피부** | 창백 (햇빛을 안 봄) + 코끝·귀끝만 약간 붉음 + **턱라인 성인 여드름** |
| **머리** | 까맣고 약간 긴 생머리, 정돈 안 됨 (떡진 머리) |
| **체형** | 키 작음, 좁은 어깨, 마른 팔다리 + **복부비만** (운동 제로 + 야식의 결과) |
| **복장** | 늘어난 **검정 후드티** + 트레이닝 바지 (항상 같은 옷) |
| **소품** | **파란 LED 전자담배** — 어두운 방에서 유일하게 빛나는 것 |

### 막별 외형 변화

| 막 | 변화 | 인상 |
|----|------|------|
| **1막** | 통통 + 복부비만 + 여드름 + 다크서클 | 찌질한 히키코모리 |
| **2막** | 거의 동일 (미세하게 야위기 시작) | 정액 소모 시작 |
| **3막** | 뚜렷하게 야윔 + 다크서클 심화 | 매일 밤 정액을 빼앗김 |
| **4막 초반** | 광대뼈 도드라짐, 꺼진 볼, 핏줄 비침 | **미라 같은 얼굴** — 동료가 걱정할 수준 |
| **4막 후반** | 소모 속도 감소, 약간 회복 | 성감 개발 성공 → 생존 안정화 |

> **배경**: 엄마가 일찍 죽고 **아버지 손에서 자람**.
> 히키코모리 생활 중 **아버지의 지원이 끊김**.
> 자신이 익숙한 분야인 **게임 콘텐츠를 만드는 작은 회사**에 입사.
> 수도에 왔지만 돈이 없어, 여성 세입자들이 연이어 자살해 소문이 퍼진 이 집이 **파격적으로 싸게** 나와 입주.

## 처녀귀신 (히로인)

### 과거
- 남편은 **귀한 독자** — 시댁에서는 손주를 간절히 바람
- 여자는 **고아** — 가족이 없었기에 **자신만의 가족을 갖는 것**이 평생의 소원
- 시댁에서 **결혼을 반대** (고아와의 결혼을 탐탁지 않게 여김)
- 제대로 된 결혼식을 올리지 못하고, **야반도주하듯** 따로 살림을 차림
- 둘이서 **소박한 예물로 약조** — 화려하진 않지만 진심을 담은 서약
- **혼인신고서도 제출하지 못함** — 법적으로는 부부가 아닌 채로 죽음
- 남편의 아이를 갖는 것은 시댁의 기대만이 아닌, **자신의 진심이기도 했음**
- **초야**를 치르기 위해 집에서 준비하고 있었음
  - 목욕을 하고 속옷을 갈아입고
  - 침대를 정리하고 향초를 켜고
  - 설렘과 긴장 속에 남편을 기다리는 중
- 같은 밤:
  - **사이코패스 방화범의 연쇄 방화** — 여자가 사는 집에 불이 남
  - 여자: **방화로 인한 질식사**
  - 남편: 집으로 오는 길에 **교통사고**로 사망 → 사고 현장 부근에 **지박령**으로 남아있음
- 사랑하는 남자의 씨를 **단 한 번도 받지 못한 채** 죽음
- 초야를 치르지 못했기에 **성교하는 법을 모름** — 본능적 욕구만 있을 뿐
- 건물은 **전소 후 재건축** — 과거의 유품은 아무것도 남아있지 않음
- 사후 집터는 **여자 전용 기숙사/임대 건물**로 운영됨
- 남자가 없는 환경에서 한이 점점 쌓임
- 급기야 여자 입주자들의 정신을 갉아먹어 **자살로 몰아감** → **악귀**로 변질

> **한의 본질**: 고아로서 가족이 없었기에, **자신만의 가족을 가지는 것** — 엄마가 되는 것
> — 인생에서 가장 행복했어야 할 밤에 **모든 것을 빼앗겼다**

### 악귀 상태 — 외형 (1막)

사람의 형태가 아니다. 성별도 알 수 없다.

| 부위 | 묘사 |
|------|------|
| **전체** | 검고 모호한 그림자. 그로테스크한 형상 |
| **눈/입** | 흔적만 — 그림자 속 어둠, 사람의 것이 아님 |
| **형태** | 불특정. 천장에서 내려오기도, 뒤에서 스치기도 함 |
| **촉감** | 비물질적. 만질 수 없음 |

### 악귀의 전략 (1막)
- 대상을 **끊임없이 놀래키고 수면을 방해**함
- 공포와 수면 부족으로 **정신력을 갉아먹음**
- 피해자의 **신체 통제력을 강탈**하여 자살을 유도 (목매달기 등)
- 이 방식으로 이전 여성 입주자들을 죽여왔음
- 물리력 행사도 가능하지만 **기운을 크게 소모**하므로 평소에는 정신 공격을 선호
  - 정신 공격이 통하지 않을 때에만 물리적 수단으로 전환

### 정화 과정 (= 잊었던 자신을 기억하는 과정)

정액의 양만큼 복원되는 것이 아니라, 정액이 **잊었던 자신의 모습을 떠올리게 하는 촉매** 역할을 한다.

| 단계 | 계기 | 변화 | 접촉 |
|------|------|------|------|
| **악귀 상태** | — | 검은 그림자, 성별 불명, 사람 형태 아님 | 비물질 |
| **첫 정액 노출** | 목을 조르다 정액 접촉 | 사람 형태를 되찾기 시작. 배꼽 주변에 온기 | 일부 실체화 |
| **반복 노출** | 매 사정마다 몰래 정액을 받음 | 여체를 가진 귀신으로 복원. 가슴, 눈 되찾음 | 귀신이 남자를 만질 수 있음 |
| **관계 심화** | 성관계 반복 | 아름다운 여성으로 완전 복원 | **서로** 만질 수 있음 |

> 정액은 **양적 연료**가 아니라 **기억의 촉매** — 맞을수록 생전의 자신을 더 많이 기억해내고, 그만큼 더 간절히 원하게 된다.

### 원래 외모 (생전 / 완전 복원 시)

| 부위 | 묘사 | 서사적 의미 |
|------|------|----------|
| **키** | 아담한 편 (크지 않음) | 고아라 성장기에 잘 먹지 못했음 |
| **어깨** | 좁은 어깨, **라운드 숄더** | 큰 가슴을 숨기려는 습관에서 비롯된 체형 |
| **가슴** | **풍만함** | 모성애의 상징. 젖 물리기와 직결 |
| **골반·엉덩이** | **풍만하게 발달** | 아이를 갖고 싶어하는 마음의 신체적 표현 |
| **아랫배** | **젖살이 덜 빠진 귀여운 느낌** | 어린 느낌과 여성스러움의 공존 |
| **전체 체형** | 마른 편은 아님. 통통하진 않지만 가슴과 엉덩이가 풍만하게 보이는 체형 | 결핍에서 벗어나 삶을 누리려는 의지 |
| **성격** | 남편에게 의존적, 순종적. 시댁에 맞서는 강단은 없음 | 남편이 사라지고 홀로 남았을 때 무너지는 이유 |

### 귀신의 의사소통 제약

| 단계 | 가능한 표현 |
|------|-----------|
| 악귀~초기 복원 | 없음 (존재만으로 공포) |
| 복원 중반 | 단편적 소리, 신음 |
| 복원 후반 | 단편적 단어 (한두 마디) |
| **성불 직전** | **자연스러운 대화 가능** — 이때 사연을 말함 |

### 불 트라우마
- 방화로 죽었기에 **불을 극도로 무서워함**
- 주인공의 평소 생활(전담, 인스턴트) 덕에 평소에는 발현되지 않음
- **불을 켜면 그날은 귀신이 나타나지 않음**

### 귀신의 호의 (관계 심화 후)
- 집 **청소**를 해줌
- 물건이 쓰러질 때 **맞지 않게** 보호해줌

## 핵심 메커니즘

### 양기와 탁기

```
남자의 정액 (양기/생명의 씨)
    ↓
악귀의 탁기를 정화
    ↓
귀신의 모습·감각·자아 복원
```

### 균형 시스템

| 상태 | 결과 |
|------|------|
| 사정 전에 성욕 충분히 해소 | 귀신 만족 → 정기 소모 최소화 |
| 성감 개발 부족 + 사정에 의존 | 정기 급격히 감소 → 복상사 위험 |
| 정기가 마이너스 | **복상사 → 사망** |

> **정력제**: 정기를 일시적으로 보충하지만 체력을 끌어다 쓰므로 1~2일 피로 부작용.
> 근본적 해결이 아닌 응급 수단.

### 성감 개발 (4막 핵심)

- 귀신의 성욕이 에스컬레이트 → 주인공이 체력 저하/위기를 느낌
- "이러다 죽겠다" → VR 야동을 구매
  - VR 콘텐츠는 **귀신과의 성감 개발과 동일한 시스템**으로 작동 (터치 영역, 리듬 판정 등 내부 로직 동일)
  - 차이: 시각적 피드백(흥분도 표시 등)이 **없을 뿐**
  - 게임은 이것이 튜토리얼이라고 알려주지 않음 (암묵적 튜토리얼)
- 귀신도 초야를 치르지 못한 처녀라 **성감을 모름** → 함께 배워가는 구조
- 성감 개발이 진행될수록 → 정액 집착 감소 → **생존 안정화**
- → 양기 고갈/복상사 위험을 낮추는 **핵심 생존 메커니즘**

## 스토리 흐름 (기승전결)

> 상세는 [scenario.md](scenario.md) 참조

### 1막: 공포 (기)
- 이사, 공포, 수면 방해 — 순수 호러
- 악귀는 검은 그림자. 사람 형태 아님
- 공포 게이지가 핵심 — 정신력을 갉아먹음
- VR 구매 → 공포 도피 + 자위 시작

### 2막: 정화의 시작 (승)
- VR로 정신공격 차단 → 악귀가 기운을 소모해 물리적으로 목을 조름 → VR 자위 중 사정한 정액에 노출
- 매 사정마다 몰래 정액을 받으며 사람 형태 복원
- 보름달: 성기를 빨다가 발각 → 벽에 숨은 귀신 엉덩이에 삽입
- 이후 귀신이 모습을 감춤

### 3막: 접근 (전)
- 귀신이 종종 보이지만 이제 악령이 아님 → 무섭지 않음
- 다가가면 도망, 팜플렛에 관심, 아이 관련 소품에 반응
- 밤에 몰래 젖 물림 + 정액 핥기 (들키면 도망)
- 가벼운 스킨십 가능 (호감도/성감대 변동)
- 보름달: 성기 핥기 → 도망 안 감 → 본격적 관계

### 4막: 성감 개발과 성불 (결)
- 성욕 에스컬레이션 → 양기 부족 → 성감 개발 시작
- 귀신이 청소를 해줌 (대화는 불가, 단편적 단어만)
- 불을 켜면 그날은 나타나지 않음 (불 트라우마)
- **IDC 방화 사건** → 귀가길에 노녀(시어머니) 구출
- 지박령(남편)이 노모를 구해준 주인공에게 **은혜를 갚으려** 집까지 따라옴
- 지박령이 귀신(아내)과 재회 → 한이 풀리며 성불 (플레이어만 목격, 주인공은 모름)
- 귀신의 성욕 증가 (성감 개발 미비 시 양기 고갈 위험)
- 귀신이 임신 → 성불 직전 사연을 말함 → 성불

## 멀티 엔딩

### 배드 엔딩

| 엔딩 | 조건 | 결과 |
|------|------|------|
| **심장마비** | 공포 수치 최대 상태에서 놀람 이벤트 발생 | 심장마비로 사망 |
| **자살** | 정신력 약화 + 체력 중간 이하 | 악귀가 신체 통제력 강탈 → 자살 |
| **복상사** | 성교 중 정기가 마이너스로 떨어짐 | 양기 소진 → 사망 |
| **악귀 회귀** | 귀신이 나온 집이 무서워 이사 나감 | 집은 다시 악귀의 집으로... (순환) |
| **양기 고갈** | 4막에서 성감 개발 실패 | 귀신의 성욕 감당 못함 → 양기 소진 사망 |

### 노멀 엔딩

| 엔딩 | 조건 | 결과 |
|------|------|------|
| **퇴마 (퇴마사)** | 핸드폰으로 진짜 퇴마사를 찾아 호출 | 귀신 퇴치 |

> **사기 퇴마사**: 핸드폰 쇼핑에서 퇴마사를 검색하면 진짜와 가짜가 섞여 있음.
> 사기 퇴마사에게 돈을 보내면 오지 않고 돈만 날림 → 퇴마 실패, 게임 계속.

### 트루 엔딩

| 엔딩 | 조건 | 결과 |
|------|------|------|
| **성불** | 지박령 성불 + 성감 개발 충분 + 보름달 성교 | 귀신 임신 → 사연을 말함 → 성불 |

> 주인공이 노녀(시어머니)를 구한 선행 → 지박령(남편)이 집까지 따라옴
> 지박령이 귀신을 보고 재회 → 한이 풀리며 성불 (플레이어만 목격)
> 이후 귀신이 주인공에게 성불 직전 사연을 가볍게 전함

## 이벤트/아이템

| 이벤트 | 트리거 | 효과 |
|--------|--------|------|
| **동료의 부적** | 남자의 안색이 너무 안 좋음 | 가벼운 차단 → 귀신이 못 나타남 → 보름달에 성욕 폭발 → **복상사 엔딩** 분기 |
| **VR** | 남자의 도피 도구 | VR 착용 중 귀신이 안 보임 + 자위 |
| **정력제** | 핸드폰 쇼핑 | 정기를 일시적으로 보충하지만 체력을 끌어다 쓰므로 1~2일 피로 부작용 |
| **IDC 방화** | 4막 중반, 같은 방화범의 연쇄 방화 | 주인공이 현장 출동 → 귀가길에 노녀 구출 |
